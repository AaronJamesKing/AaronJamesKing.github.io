Experience

Over the past 7 years, I’ve worked on a wide range of custom software for over a dozen organizations. I know how to design, architect, program, debug, and maintain high-quality systems. I also understand the human aspect of software – the importance of trust, communication, and compassion when building real applications that real people rely on.

Work

Here’s a sample of what I’ve built and achieved for clients over the years:

  • A system to deliver and run firmware updates to software embedded in Intel NUCs.
  • Messaging queue for a distributed system, along with an admin dashboard to observe system state.
  • A system using TypeScript and JSON allowing product owners to update the complex, interactive content for a mobile app, without having to publish App Store or Play Store updates.
  • A report of low-level benchmark testing to compare rendering and touch detection for iOS and Android.
  • Integrate Stripe into an existing payment system to support direct purchases and subscription purchases.
  • Application updates through multiple versions of both Android and iOS, to prepare a release for a yearly event.
  • Allow a gradual migration from a multi-page legacy app to its replacement SPA using custom React components that make HTTP requests and then rendering the content.
  • Realtime A/V capture and streaming from a Windows application to a Linux application.
  • Serverless syncing for the state of a mobile app, with multiple live devices and unstable network connections.
  • Update a web app to meet accessibility requirements for the Michigan State Department of Education.
  • Refactor a large and complicated domain model in order to define and sell an exponentially larger number of products.
  • Built iOS application for capturing audio and video in order to demo client’s SDK that provides secure cloud storage.
